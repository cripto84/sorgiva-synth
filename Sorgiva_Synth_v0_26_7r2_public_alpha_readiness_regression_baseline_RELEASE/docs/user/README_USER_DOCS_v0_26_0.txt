Sorgiva Synth — User Documentation Index v0.26.0

Questa cartella contiene la prima base documentale utente della mainline desktop/core.
Non è ancora il manuale PDF finale: serve a fissare contenuti corretti, controllabili e coerenti col codice.

Documenti inclusi:
1. QUICK_START_v0_26_0.txt
   Avvio rapido: sbloccare audio, caricare preset, suonare, salvare/esportare.

2. USER_GUIDE_FOUNDATION_v0_26_0.txt
   Guida generale alle schede della UI e al flusso mentale di Sorgiva Synth.

3. SOUND_DESIGN_GUIDE_v0_26_0.txt
   Guida pratica alla costruzione dei timbri: oscillatori, PWM, Ring, FM, Sync, Unison, filtri, inviluppi, FX.

4. FACTORY_PRESET_LIBRARY_GUIDE_v0_26_0.txt
   Stato reale della factory library rebuild: 135 preset, gruppi musicali e categorie browser, inclusi Filter Bass e Guitar-like / Synth Guitars.

5. PRESET_PATCH_RUNTIME_BOUNDARY_v0_26_0.txt
   Differenza fra preset sonoro, patch JSON, preferenze runtime/UI, MIDI runtime e visualizer.

6. ARPEGGIATOR_STEP_SEQUENCER_GUIDE_v0_26_0.txt
   Uso di Arpeggiator, Behavior Macro, Motion Shape interna e Step Sequencer senza confonderli.

7. MIDI_PERFORMANCE_GUIDE_v0_26_0.txt
   Tastiera, velocity, sustain, pitch bend, mod wheel, aftertouch/expression/breath/foot, MIDI Learn e MIDI Clock.

8. RANDOMIZER_MORPH_AB_MODMATRIX_GUIDE_v0_26_0.txt
   Strumenti di esplorazione sonora: Randomizer controllato, A/B Compare, Morph e Mod Matrix.

9. VISUALIZERS_AND_KNOWN_LIMITS_v0_26_0.txt
   Oscilloscopio, spettroscopio, limiti noti, test reali ancora pending.

Stato tecnico dichiarato:
- factory preset count: 135;
- vecchia factory da 290 preset: ritirata e non reintrodotta;
- JSON storici sotto presets/: non reintrodotti;
- audio-context.js: non modificato in questa fase;
- documentazione lunga: confinata sotto docs/.
