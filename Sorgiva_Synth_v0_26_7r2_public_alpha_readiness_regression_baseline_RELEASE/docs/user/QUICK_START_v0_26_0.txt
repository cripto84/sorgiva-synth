Sorgiva Synth — Quick Start v0.26.0

1. Apri `index.html` in un browser moderno.
   Sorgiva Synth è una web app locale: non serve un server per la prova base.

2. Premi `Sblocca Audio`.
   I browser bloccano l'audio finché non c'è un gesto dell'utente. Senza questo passaggio potresti vedere la UI funzionare ma non sentire suono.

3. Suona dalla tastiera virtuale in basso, dalla tastiera PC se supportata dalla build, o da MIDI.
   La tastiera virtuale resta visibile in basso ed è pensata come riferimento sempre disponibile.

4. Carica un factory preset.
   Vai in `Preset & Patch`, scegli categoria/filtro, seleziona un preset e premi `Carica preset factory`.
   La factory rebuild attiva contiene 135 preset.

5. Modifica il suono.
   Flusso consigliato:
   - `Oscillatori & LFO`: sorgenti, PWM, Ring Mod, FM Light, Osc Sync, Unison;
   - `Filtri & Inviluppo`: HPF/BPF/Notch/VCF, Filter ADSR, Filter Drive, Advanced Filter;
   - `Equalizzazione` e `Effetti`: EQ, Drive/Saturation, modulazione, delay, reverb;
   - `Mod Matrix`: collegamenti espressivi/modulanti avanzati.

6. Usa Arp o Step Sequencer solo se vuoi movimento automatico.
   La Motion Engine mantiene Manuale / Arp / Sequencer separati: evita di trattare l'arp come un secondo step sequencer.

7. Salva il risultato.
   - `Salva copia utente` crea una copia locale di un factory preset modificato.
   - `Export JSON` esporta una patch/preset in file.
   - `Copia JSON factory` o `Export JSON factory` sono strumenti tecnici, non modificano la factory sorgente.

8. Usa `Panic`/Stop se qualcosa resta in sospeso.
   Sorgiva Synth ha guardrail anti-stuck, ma durante test MIDI/browser è sempre utile avere Panic a portata.

Nota importante:
Caricare un preset sonoro non deve spegnere oscilloscopio/spettroscopio, non deve modificare MIDI Learn, MIDI Clock, channel filter o preferenze runtime/UI. Questo confine è stato blindato nella linea v0.25.16a/v0.25.17 e documentato in v0.26.0.
