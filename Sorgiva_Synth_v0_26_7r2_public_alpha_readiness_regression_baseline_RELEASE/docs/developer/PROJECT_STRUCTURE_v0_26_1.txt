Sorgiva Synth — Project Structure v0.26.1

Root
- index.html
  UI principale, schede, controlli, tastiera virtuale, link CSS/JS.
- README.txt
  Sommario della build corrente e stato operativo.
- CHANGELOG.txt
  Registro sintetico delle versioni recenti.
- VERSION
  Stringa versione macchina.
- VERSION.txt
  Titolo versione leggibile.
- src/
  Codice JavaScript modulare.
- styles/
  CSS della UI.
- presets/
  Cartella per preset esterni/strutturali. In questa linea non deve contenere JSON storici confusi.
- docs/
  Documentazione storica, utente, sviluppatore e release notes.

src/audio/
- audio-config.js
  Default, liste parametri e costanti condivise: oscillatori, filtri, FX, sequencer, arp, visualizer, safety, tuning.
- audio-dsp.js
  Funzioni DSP pure/di supporto: clamp, tuning, conversioni MIDI/frequenza, curve, utilità audio.
- audio-context.js
  Motore WebAudio: context, voice management, oscillatori, filtri, inviluppi, FX, safety, note on/off, routing e onParameterChange.

src/state/
- state.js
  Stato globale, parametri, snapshot, subscribe, update di sottostati runtime come performance, motion, visuals, MIDI, tuning.

src/ui/
- controls.js
  Binding dei controlli HTML, update label/status, refresh UI, routing di cambi parametro verso SynthXState.
- tabs.js
  Gestione schede ARIA/tab UI.

src/keyboard/
- keyboard.js
  Tastiera virtuale, mapping note, interazione mouse/tastiera e inoltro note al motore.

src/arp/
- arpeggiator.js
  Arpeggiatore, held notes, latch, behavior macros, motion shapes, timing, gate, panic, export/import macro.

src/sequencer/
- step-sequencer.js
  Step Sequencer a 16 step con active/note/octave/velocity/gate/accent/tie, pattern preset, export/import.

src/motion/
- motion-engine.js
  Stato esclusivo Manuale / Arp / Sequencer e prevenzione conflitti fra generatori di movimento.

src/midi/
- midi.js
  MIDI input, note, pitch bend, mod wheel, aftertouch, expression, breath, foot, sustain e channel filter runtime.
- midi-clock.js
  MIDI clock, BPM esterno/interno e sincronizzazione dove supportata.
- midi-learn.js
  MIDI Learn e mapping controlli.

src/modulation/
- modulation-matrix.js
  Gestione Mod Matrix a 8 slot, normalizzazione sorgenti/destinazioni e aggiornamento stato runtime.

src/presets/
- preset-constants.js
  Versione formato preset, storage keys, ID esclusi, runtime boundary, tassonomia factory.
- factory-presets.js
  Libreria factory attiva corrente da 135 preset. In v0.26.1 era a 111; dopo v0.26.3 include anche Filter Bass e Guitar-like / Synth Guitars.
- presets.js
  Preset manager, factory browser, user bank, import/export, normalizzazione, runtime boundary.
- randomizer.js
  Randomizer controllato con guardrail musicali.
- preset-morph.js
  Preset Morph e A/B Compare.

src/visuals/
- oscilloscope.js
  Oscilloscopio canvas runtime. Preferenza UI, non parametro sonoro preset.
- spectroscope.js
  Spettroscopio canvas runtime. Preferenza UI, non parametro sonoro preset.

styles/
- main.css
  Stile generale. Mantenere leggibile, non comprimere la tastiera, non stravolgere layout.

docs/
- user/
  Documentazione utente fondativa introdotta in v0.26.0.
- developer/
  Documentazione sviluppatore fondativa introdotta in v0.26.1.
- release-notes/
  QA report, metriche, release notes e storico riordinato.
