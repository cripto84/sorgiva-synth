Sorgiva Synth — Known Limits
=============================

Current known limits
--------------------

- A real-controller MIDI baseline passed on 2026-07-09: note on/off, velocity, Mod Wheel and responsive input with no perceived lag. This is not a guarantee for every controller, operating system or browser.
- Extended browser listening QA is still required for final public alpha qualification.
- Browser CPU benchmark coverage is still incomplete, especially for 12-layer Unison, long releases, complex chords and heavy FX chains.
- Firefox does not provide the same WebMIDI path as Chromium and remains outside the validated MIDI baseline.
- Mobile/tablet is not part of the desktop/core mainline.
- Sorgiva Synth is not a DAW and does not aim to replace a full DAW.
- The Step Sequencer and Arpeggiator are performance/motion tools, not full composition systems.
- Factory presets are a curated rebuilt library of 135 presets, not the retired 290-preset legacy library.
- Documentation is text-first at this stage; final PDF/user manual packaging remains a later editorial pass.

Audio safety notes
------------------

Sorgiva Synth includes gain guard and conservative defaults, but users should still start with low monitoring volume,
especially when experimenting with FM, Ring Mod, Osc Sync, Unison/Detune, feedback delay and resonant filters.
